# Enrollment Management -- User Stories

## List and Search

- As an operator, I want to see all enrollments so I can monitor learner activity.
- As an operator, I want to filter enrollments by status so I can find active/completed/withdrawn learners.
- As an operator, I want to search enrollments by learner name or email so I can find a specific person.
- As an operator, I want to filter by release version so I can see who is on which content version.

## Detail View

- As an operator, I want to view an enrollment's detail page so I can see the learner's full progress.
- As an operator, I want to see module-by-module progress so I can identify where a learner is stuck.
- As an operator, I want to see FAA-qualified vs total time so I can verify compliance.
- As an operator, I want to see lesson attempt history so I can understand a learner's journey.

## Create

- As an operator, I want to create a new enrollment for a learner so they can begin the course.
- As an operator, I want to select which published release to enroll against so the learner gets the correct content version.
- As an operator, I want to be prevented from double-enrolling a user in the same release so records stay clean.

## Status Management

- As an operator, I want to mark an enrollment as completed so the learner can receive a certificate.
- As an operator, I want to withdraw an enrollment so I can handle drops or policy violations.
- As an operator, I want status changes to be audited so there's a record of who changed what and when.
