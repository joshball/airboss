# Reflection

Products for after flying. Journals, heatmaps, trends, calibration -- tools that help a pilot see patterns in their own performance over time.

Most pilots debrief informally or not at all. These products make reflection structured and visible, turning "I think I'm getting better" into "here's what changed and why."
