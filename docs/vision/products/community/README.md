# Community

Products with a social dimension. Buddies, shared scenarios, CFI pairing, anonymous sharing -- features that connect pilots to each other.

Aviation already has strong community instincts (fly-ins, EAA chapters, flight school culture). These products give that energy a digital home, focused on learning and accountability rather than social media engagement.
