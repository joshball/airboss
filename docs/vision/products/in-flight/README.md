# In-Flight

Products used during or immediately after a flight. Cockpit reference cards, real-time prompts, debrief capture -- tools that live in the airplane or the car ride home.

Constraint: minimal interaction. A pilot in the cockpit has no attention to spare. These products must be glanceable, tap-once, or voice-driven. Nothing that requires reading paragraphs at 3,000 feet.
