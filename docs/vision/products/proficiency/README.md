# Proficiency

Products for between flights. Daily practice, skill maintenance, deliberate drills -- the work that keeps a pilot sharp when they're not in the airplane.

Currency is the FAA minimum. Proficiency is the actual standard. These products close the gap between "legally allowed to fly" and "actually ready to fly" through short, focused practice sessions at a desk or on a phone.
