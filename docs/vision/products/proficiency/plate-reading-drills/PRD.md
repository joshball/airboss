# Plate Reading Drills

## What it does

Flash a section of an approach plate, ask a specific question -- MDA? Missed approach turn? Non-standard minimums? Builds the chart-reading speed that separates smooth instrument pilots from ones who fumble in the soup.

## Core features

- Timed plate-reading drills: see the plate section, answer the question
- Progressive difficulty: full plate -> partial plate -> memory recall
- Covers IAPs, SIDs, STARs, and airport diagrams
- Tracks reading speed and accuracy by plate element type
- Spaced repetition on elements you get wrong

## Notes

Approach plate data is available through FAA sources but rendering and highlighting sections requires image processing work. The medium complexity comes from building a good drill engine, not from the content itself. Huge value for instrument pilots who don't fly approaches regularly.
