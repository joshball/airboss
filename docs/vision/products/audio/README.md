# Audio

Products for earbuds. Commute-friendly scenarios, drills, and stories that work without a screen -- passive learning while driving, walking, or doing chores.

Aviation is a storytelling culture. Hangar flying, NTSB reports, "there I was" tales -- pilots learn through narrative. Audio products tap into that tradition and turn dead time into study time.
