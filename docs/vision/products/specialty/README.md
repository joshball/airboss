# Specialty

Products for niche audiences. Helicopter, glider, aerobatics, sim community, returning pilots -- groups with specific needs that the general products don't fully serve.

These aren't separate apps. They're modes or content packs that adapt the core platform to a particular pilot's world. A helicopter pilot's "memory items" and "approach rehearsal" look very different from a fixed-wing pilot's.
