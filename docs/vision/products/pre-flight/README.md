# Pre-flight

Products you use before you fly. Route preparation, airport familiarization, weather review, mental loading -- everything between "I'm going flying tomorrow" and engine start.

The thread connecting these: reducing surprise. A pilot who has mentally walked the route, reviewed the airport, and rehearsed the approach arrives at the airplane already ahead of the flight.
