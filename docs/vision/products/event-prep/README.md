# Event Prep

Products for milestone preparation. BFR, IPC, checkride, type rating, interview -- any high-stakes event with a known date and defined standards.

These are sprint-mode products. A pilot has a thing coming up, knows the date, and needs a focused plan to be ready. Each product compresses the relevant knowledge and practice into a time-boxed preparation arc.
