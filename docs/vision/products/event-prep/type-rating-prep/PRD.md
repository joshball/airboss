# Type Rating Prep

## What it does

Type-specific prep for CRJ, 737, A320, Citation, King Air, and other common types. Memory items, performance profiles, callout flows, and gotchas from pilots who've been through the training.

## Core features

- Per-type memory item sets with spaced repetition
- Callout flow practice for normal and abnormal procedures
- Performance profile quick-reference (V-speeds, limitations, systems)
- "Gotchas" library from type-experienced pilots
- Mock oral questions specific to each type rating

## Notes

High complexity because each type requires deep, accurate technical content -- errors in memory items or limitations are dangerous. Content sourcing requires partnerships with type-rated pilots or training providers. Low personal need but high career-track value.
