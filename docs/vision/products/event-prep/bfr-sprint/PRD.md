# BFR Sprint

## What it does

A 4-week structured prep program for a flight review. Covers ground topics, decision scenarios, oral question bank, and memory items. Ends with a confidence score so you know whether you're ready or need more time.

## Core features

- 4-week schedule: regulations week 1, weather/performance week 2, scenarios week 3, review week 4
- Daily 10-15 minute sessions covering ground knowledge and decision practice
- Oral question bank organized by the topics CFIs actually ask about
- Progress dashboard with confidence scoring per topic area
- "Ready / not ready" assessment at the end with specific weak areas flagged

## Notes

Depends on Spaced Memory Items (prd:prof:spaced-memory-items) and Decision Reps (prd:prof:decision-reps) for core drill mechanics, and Recency Recovery (prd:evt:recency-recovery) for returning-pilot onramp. This is the highest-value event-prep product because every pilot needs a flight review every 24 months and most show up unprepared.
