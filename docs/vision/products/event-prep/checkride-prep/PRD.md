# Checkride Prep

## What it does

ACS-aligned prep for PPL, IR, CPL, CFI, and ATP checkrides. Scenario-based oral question practice organized by ACS task codes, not rote quiz format. Builds the kind of understanding DPEs are actually looking for.

## Core features

- Per-certificate prep tracks aligned to current ACS/PTS standards
- Scenario-based oral questions, not multiple-choice trivia
- Organized by ACS area of operation and task code
- Weak-area identification with targeted drill recommendations
- "DPE perspective" notes on common failure points per task

## Notes

Depends on the scenario and memory item engines. Content must track ACS updates -- FAA revises these periodically. The opportunity is that most checkride prep tools are rote memorization; scenario-based oral practice is what actually prepares you for a DPE conversation.
