# Ops Roadmap

## Current Work

**Phase 3 -- Ops Foundation** `[DONE]`

See [features/PHASE3.md](features/PHASE3.md) for the phase index.

## Phase 3 -- Ops Foundation

Operations and compliance. The first ops release.

**Features:**

1. Ops app shell (auth, nav, layout, role guards)
2. User management (accounts, roles, invite/ban)
3. Enrollment management (create, view, manage)
4. Learner progress view (modules, time, scores)
5. Certificate issuance (graduation vs completion, PDF)
6. FAA records (24-month retention, evidence packets, audit trail)
7. Analytics dashboard (completion rates, struggle points)

**Exit criteria:** An operator can manage users, view learner progress, issue certificates, and pull FAA records.

## Phase 4 -- Ops Enrichment (Future)

- Advanced analytics (cohort comparison, trend detection)
- Compliance dashboard (traceability matrix, FAA submission workflow)
- Bulk operations (batch enrollment, group certificate issuance)
- Report generation (compliance package, instructor records)
- TSA record management (5-year retention per 49 CFR 1552)

## Phase 5+ (Future)

- External integrations (FAA IACRA, TSA flight training API)
- Multi-provider support (multiple FIRCs managed from one ops)
- Custom report builder
