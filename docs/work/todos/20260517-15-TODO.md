## Not in scope (182 remaining warnings -- non-handbook corpora)

The remaining 182 `legacy-citation-shape` warnings are non-handbook corpora:
AIM (~47), AC (~62), CFR / regs (~20), ACS / PTS (~11), POH / AFM (~14),
NTSB (~6), FAA Orders (~2), FAA pamphlets (FAA-P / FAA-S), plus a handful of
non-FAA sources (AOPA ASI, Jeppesen, drs.faa.gov).

These cannot be migrated to `ref:` today. The `@ab/sources` validator's rule 2
ERRORs on any `airboss-ref:` whose corpus has neither a registry entry nor an
`isKnownLocator` resolver predicate. Only the handbooks resolver implements
`isKnownLocator`; every other corpus is registry-backed and the registry ships
empty (ADR 019 Phase 2). Converting an AIM / AC / regs `source:` to `ref:`
today turns a non-blocking WARNING into a build-blocking ERROR.

Migrating those corpora is the ADR 019 Phase 3 (CFR ingestion), Phase 7 (AIM),
Phase 8 (AC), and Phase 10 (irregular corpora) work -- registry population, not
a citation-shape rewrite. This is surfaced as a decision in the PR body.
