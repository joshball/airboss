## Hangar course editor (course-reader-and-editor, post-review)

**URL**: `https://hangar.airboss.test/courses`

**How to test manually**:

1. Sign in, go to `/courses` in the hangar app.
2. Open a course, then a section editor.
3. Drive the KnowledgeNodePicker with the keyboard only: ArrowDown opens the
   listbox, Enter selects, Escape closes. This was a Critical a11y fix (PR #1042).
4. Try creating a course with an invalid slug (e.g. containing `../`) -- it must
   be rejected server-side (Critical security fix, PR #1042).

**What to look for**: keyboard-only node picking works; path-traversal slugs are
rejected; the section editor shows sections (a `process.cwd()` regression was
fixed in PR #1042).
