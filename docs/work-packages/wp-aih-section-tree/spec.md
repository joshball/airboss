# WP-AIH -- Aviation Instructor's Handbook section-tree

Promote AIH (FAA-H-8083-9) from the whole-doc handbooks-extras path to the chapter-aware `handbooks/<slug>.yaml` pipeline. Third book in the whole-doc-promotion sweep (after mtn-tips and RMH).

Source spec: [whole-doc-promotion/research.md §AIH](../whole-doc-promotion/research.md#aih----aviation-instructors-handbook-faa-h-8083-9).

## Goal

Land AIH as a chapter+section drill-down `/library` card backed by per-chapter PDFs and the embedded outline (551 entries, depths 1-5). Replace the single whole-doc manifest with a manifest that carries `sections[]` rows for chapters, sections, and (depth-3) subsections.

## Source

- Publisher index: <https://www.faa.gov/regulations_policies/handbooks_manuals/aviation/aviation_instructors_handbook>
- Whole-doc PDF: `<index>/aviation_instructors_handbook.pdf` (228 pages)
- Per-chapter PDF pattern: `<index>/<NN>_aih_chapter_<N>.pdf`, N = 1..10
- Ancillary PDFs (front=cover+ack+preface, 4 appendices, glossary, index)
- Edition: `FAA-H-8083-9` (FAA index page lists publication date 2020; no revision letter)
- Embedded outline: 551 entries, depths {1: 18, 2: 155, 3: 246, 4: 129, 5: 3}

## Decisions (taken at WP author time)

- **Class**: A2 (per-chapter PDFs published, separately-distributed ancillaries). Mirrors AFH.
- **`outline_strategy: bookmark`**. The PDF outline is rich and clean (chapter titles include the canonical "Chapter N: ..." prefix, section titles render directly). Same shape as AVWX.
- **`section_strategy: toc`**. Kept for symmetry with the other chapter-aware handbooks; the bookmark tree is the source of truth for the section rows.
- **L4/L5 flatten**: 132 depth-4 entries and 3 depth-5 entries collapse to depth-3 per the existing `parse_outline` cap. Spot-check chapter 1 ("Use of Resources" subdivisions) and chapter 9 to confirm the flatten is acceptable. Flag if not.
- **`subjects`**: `['training-ops', 'human-factors']` (matches the current handbooks-extras row).
- **`primary_cert`**: `cfi` (matches the current handbooks-extras row; surfaced through the YAML for the existing register pipeline).
- **`file_ordinal_offset`**: 2 (cover=00, acknowledgments=01, preface=02, ch1=03 ... ch10=12, appA=13, appB=14, appC=15, appD=16, glossary=17, index=18).
