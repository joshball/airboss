# User Stories: Spaced Memory Items

## Studying

- As a returning pilot, I want to create flashcards as I read the FARs so that creating the card IS the studying -- I learn by writing, not just reading.
- As a pilot, I want to review due cards in a focused session so that I retain aviation knowledge through spaced repetition rather than cramming.
- As a pilot, I want the system to schedule my reviews so that I see cards just before I'd forget them -- not too early (wasted effort), not too late (forgotten).
- As a pilot, I want to rate how well I knew each answer so that the algorithm adjusts future scheduling to my actual performance.

## Organization

- As a pilot studying multiple domains, I want to tag cards by domain (regulations, weather, airspace) so that I can see which areas I'm strong and weak in.
- As a pilot, I want to browse and filter my card deck so that I can find, edit, or review specific cards.
- As a pilot, I want to suspend a card temporarily so that I can focus on other material without losing the card's review history.

## Progress awareness

- As a pilot, I want to see how many cards are due today so that I know the size of my study commitment before starting.
- As a pilot, I want to see my review streak so that I stay motivated to study daily.
- As a pilot, I want to see per-domain mastery stats so that I know where to focus -- "I'm solid on airspace but weak on weather."

## Calibration (cross-feature)

- As a pilot, I want to predict my confidence before seeing the answer so that I can discover where I'm overconfident or underconfident.

## Future integration

- As a course learner (future), I want course-assigned cards to appear in my review queue alongside personal cards so that I have one study flow, not two.
- As a pilot using Route Walkthrough (future), I want the system to automatically create review cards from route-specific knowledge so that pre-flight learning becomes long-term retention.
