# Week 8 drills

Week 8 lessons mapped the regulatory ecosystem outside the FAR: the AIM (expected practice), Advisory Circulars (acceptable means), Chief Counsel letters (binding interpretation), 49 CFR (cross-agency rules), and FAA Orders (internal procedure). The drills exercise a single skill: given a regulatory question, name the document that owns the answer.

This is the source-triage drill. The pilot does not need to *answer* the question (that's a different skill); they need to name *where the answer lives*. Working pilots and CFIs route questions to documents in seconds; the drill builds that routing instinct.

These drills feed `apps/study/` flash cards. Each prompt is a single card; the answer card names the document type and the specific reference.

## Hardcover drill -- 60 seconds, no notes

Integration. Read the prompt, give the answer in 60 seconds without referring to anything.

1. **Name the five document layers and what each is.** -> FAR (Layer 1, binding regulation, 14 CFR). Cross-title CFR (Layer 2, 49 CFR primarily, NTSB / TSA / BasicMed statutory). AIM (Layer 3, expected practice, enforceable through §91.13). Advisory Circulars (Layer 4, one acceptable means of compliance). Chief Counsel letters (Layer 5, binding interpretation of contested cases). FAA Orders sit alongside as agency procedure.
2. **A pilot asks if they can split fuel costs with a friend going to a wedding. Walk me through the answer.** -> §61.113(a) prohibits compensation for a private pilot. §61.113(c) carves out pro-rata cost-sharing for "a common purpose." The Mangiamele letter (2009) holds that "common purpose" requires the pilot have an independent reason for the flight. So: was the pilot going anyway? If yes, common purpose is satisfied and the pro-rata split is legal. If no -- if the flight is happening *because* the friend asked -- it's transportation for hire and prohibited.
3. **The radio fails halfway to the Class C airport you're going to. Walk me through what to do, citing sources.** -> AIM 6-4 + Pilot/Controller Glossary on light gun signals. §91.129 establishes the requirement for two-way comm in Class C; you can't comply directly, so you need to (a) attempt to reestablish on the assigned freq or 121.5, (b) squawk 7600, (c) approach the airport at altitudes that put you in line with the pattern, (d) watch for light gun signals, (e) follow the pattern as if cleared. Chapter 4-3 of the AIM gives the pattern and lights detail.
4. **A new student is a Brazilian national. Can you start training tomorrow?** -> No. 49 CFR Part 1552 (TSA AFSP) requires the student be vetted before training begins. The CFI initiates the application (or the school's AFSP coordinator does). Training waits for TSA approval.
5. **Walk me through finding the authoritative answer to a regulatory question you've never seen before.** -> Read the question, find the FAR, read the FAR's neighborhood. If procedural, check AIM. If the FAR says "in a manner approved by the Administrator," check the AC. If contested, search Chief Counsel letters by section number. If cross-agency (security, reporting, medical statutory), check 49 CFR. If about agency procedure, check FAA Orders. Synthesize the answer with the legal frame (FAR), the operationalizing guidance (AIM/AC), and the controlling interpretation (CC letter) where applicable.

## Where this drill set fits

These drills are the connective tissue from the Week 8 lessons to `apps/study/`. Format 1 (the main "where does it live?" drill) -> source-triage cards. Format 2 (signal recognition) -> meta-cards on routing discipline. Format 3 (famous letters) -> Chief Counsel cards. Format 4 (high-yield ACs) -> AC cards. Format 5 (AIM chapters) -> AIM map cards.

The drill set is the practice instrument for the Week 8 oral. The oral integrates the routing skill into a checkride-style scenario.
