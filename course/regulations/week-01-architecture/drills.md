# Week 1 drills

The week 1 drill set has three formats, each exercising a different skill from the foundation lessons:

1. **Locate** -- given a question, name the part / subpart / section
2. **Parse** -- given a citation (written or spoken), produce the precise written form
3. **Route** -- given a question, name the three-subject category and where to look

These map directly to flash cards in `apps/study/`. The card type is **navigation prompt** in each case -- the front shows the prompt, the back shows the citation or the routing.

### Cross-cutting questions

These hit multiple subjects. Walk them in the order specified.

| Question                                                                  | Sequence |
| ------------------------------------------------------------------------- | -------- |
| "I'm flying IFR at night with a passenger tomorrow."                      | Pilot (61) -> Flight (91) |
| "My CPL student is leaving to fly for a 135 operator."                    | Pilot (61) -> Operation (135 + 119) |
| "I'm a CFI giving a flight review in the student's club airplane."        | Pilot (61) -> Flight (91) |
| "Can I send my 141 student on a solo cross-country in marginal weather?"  | Pilot (61.93) -> Flight (91.155) -> Operation (141 syllabus mins) |
| "Student lands gear-up at night IFR. Walk me through what happens."       | Flight (91.3, 91.7, 91.405) -> Operation (none here, this is 91) -> Reporting (49 CFR 830) |
| "My friend asks for a flight review. Walk me through what I check."       | Pilot (61.56, 61.193, 61.195, AC 61-65) -> Flight (91 for the actual flight) |
| "PPL with 250 hours wants to start IR training."                          | Pilot (61.65 eligibility) -> Operation (141 vs 61 school choice) -> Pilot (specific currency) |

## Format 4 -- Trap detector

For each statement, identify the misreading and correct it. (Cover the right column.)

| Statement                                                                 | Trap and correction |
| ------------------------------------------------------------------------- | ------------------- |
| "Part 91 only applies to non-commercial flying."                          | Wrong. Part 91 applies to *every* civil flight in U.S. airspace. 135 / 141 / 121 stack on top. |
| "The AIM is the regulation for ATC procedures."                           | Wrong. The AIM is *expected practice*, not regulation. Deviation can support a §91.13 charge but the AIM itself is non-binding. |
| "Subchapter D is just Part 61."                                           | Wrong. Subchapter D includes Parts 60, 61, 63, 65, 67, 68. Pilot cert is 61; medical is 67; BasicMed is 68. |
| "NTSB rules are in Title 14."                                             | Wrong. NTSB rules live in Title 49 (49 CFR 830). |
| "ATOMATOFLAMES is the equipment list."                                    | Wrong. It's the VFR-day list only. Add (c) for VFR night and (d) for IFR. |
| "Ninety-one point one-oh-three"                                           | Wrong. The period is a separator, not a decimal. Read as "ninety-one one-oh-three." |
| "AC 61-65 requires this endorsement."                                     | Wrong. ACs are guidance. The endorsement is required by §61.31 / §61.93 etc.; AC 61-65 describes the recommended language. |
| "I'm at 61.13 for the careless rule."                                     | Wrong. §61.13 is "Issuance of airman certificates." The careless rule is §91.13. Different parts entirely. |
| "Part 135 replaces Part 91 for charters."                                 | Wrong. Part 135 *adds* requirements on top of Part 91. Both apply. |
| "Part 67 covers BasicMed."                                                | Wrong. Part 67 covers the FAA medical examination process. BasicMed lives in Part 68 with statutory basis in 49 USC 44703. |

## Hardcover drill -- 60 seconds, no notes

The integration drill. Read the prompt, give the answer in 60 seconds without referring to anything.

1. **Where in Title 14 does pilot certification live, and what's the corresponding medical Part?** -> Part 61 for cert; Part 67 for medical (Part 68 for BasicMed).
2. **Read aloud and parse: §91.213(d)(2)(ii).** -> "Section ninety-one two-thirteen paragraph d sub-paragraph two item ii." (Part 91, Section 213, paragraph d, sub-paragraph 2, item ii.)
3. **A 200-hour private pilot wants to take three friends at night to KSBA tomorrow IFR. Name the subject categories and the order to walk.** -> Pilot first (61.57 currency, 61.56 flight review, Part 67 medical). Then flight (91.103 preflight, 91.205 equipment, 91.211 oxygen if applicable, 91.167/.169 fuel + alternate, 91.409/.411/.413 inspections). Operation: not applicable; this is a 91 flight.
4. **Where do you look to find what 91.103's "all available information" actually requires?** -> Read §91.103 first. Then the Walker letter (Chief Counsel, 2017). Then AC 91-92 for what constitutes a complete preflight briefing.
5. **A pilot has a runway excursion that doesn't damage anything but ATC asks them to call the FSDO. What regulations apply?** -> 49 CFR Part 830 first (was this an accident or serious incident requiring NTSB notification?). Then §91.13 if FAA pursues a careless or reckless action. Then PBR2 protections (Pilot's Bill of Rights). Possibly the FAA's Compliance Program (Order 8000.373) instead of legal enforcement.

## Where this drill set fits

Drills are the connective tissue between the lessons and `apps/study/`. The Format 1 (Locate) drills become flash cards. The Format 2 (Parse) drills become written-recall cards. The Format 3 (Route) drills become scenario cards. The Format 4 (Trap detector) drills become misconception-correction cards.
