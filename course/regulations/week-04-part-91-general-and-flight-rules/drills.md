# Week 4 drills

Week 4 lessons walked the spine of Part 91 subparts A and B: §91.3 / §91.13 (the bookends), §91.7 / §91.103 (airworthiness and preflight), the occupant and right-of-way cluster (§§91.105, 91.107, 91.111, 91.113, 91.115, 91.119), the VFR weather and altitude cluster (§§91.151, 91.155, 91.157, 91.159), the IFR ops cluster (§§91.167-91.185), and the supplemental equipment rules (§91.211, §91.215). The drills exercise three skills:

1. **Locate** -- given a Part 91 question, name the section in seconds
2. **Apply** -- given a scenario, identify the operative rule and the answer
3. **Distinguish** -- separate similar-sounding rules without slipping (§91.105 vs. §91.107, §91.169 vs. §91.175, etc.)

These drills feed `apps/study/` flash cards. Format 1 -> navigation cards. Format 2 -> scenario cards. Format 3 -> misconception-correction cards. Format 4 -> integration cards.

## Format 3 -- Distinguish (separate similar-sounding rules)

For each pair, state the difference. Cover the right column.

| Pair                                                    | Distinguishing point                                                                                                                       |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| §91.105 vs. §91.107                                     | §91.105 = required flight crewmembers, en route + takeoff/landing. §91.107 = every person on board, taxi/takeoff/landing only.             |
| §91.169 vs. §91.175                                     | §91.169 = filing the alternate (1-2-3 rule, 600-2/800-2 alternate minimums). §91.175 = flying the approach (descent below DH/MDA).         |
| §91.119(a) vs. §91.3(b)                                 | §91.119(a) = altitude floor for normal ops (must be high enough to land without undue hazard). §91.3(b) = deviation in actual emergency.   |
| §91.155 vs. §91.157                                     | §91.155 = basic VFR weather minimums table. §91.157 = special VFR (ATC clearance, day-only default, 1 SM, clear of clouds).                |
| §91.151 vs. §91.167                                     | §91.151 = VFR fuel (30/45 min reserve). §91.167 = IFR fuel (destination + alternate + 45 min).                                             |
| §91.155 ceiling vs. cloud clearance                     | Ceiling = lowest broken/overcast layer. Cloud clearance = vertical or horizontal distance from any cloud. Different concepts.              |
| §91.211 12,500-14,000 vs. 14,000+                       | 12,500-14,000: required flight crew oxygen after 30 min. 14,000+: required flight crew oxygen at all times (no grace).                     |
| §91.215 Class B veil vs. above 10,000 MSL               | Mode C veil = 30 NM around Class B primary, surface to 10,000 MSL. Above 10,000 MSL = anywhere (with 2,500 AGL exception).                 |
| §91.113 right-of-way category order vs. right-side rule | Category order trumps right-side rule. Glider beats Cherokee on either side.                                                               |
| §91.13 careless vs. reckless                            | Careless = failure to use ordinary care (negligence). Reckless = conscious disregard (intent or near-intent). Same section, different bar. |

## Format 4 -- Integration (multi-rule scenarios)

These are the Week 6 integration drills in miniature. State every operative rule.

1. **VFR cross-country, marginal weather.** KPAO -> KMRY, forecast 2,500 broken at KPAO clearing to broken layer at 6,000 along the route, marginal visibility 4-5 SM. 172, 36 gallons, 1.4 hour flight, day. Operative rules: §91.103 preflight (all available info, including TAFs and NOTAMs), §91.151 day fuel reserve, §91.155(c) Class D departure ceiling, §91.155(a) en-route cloud clearance, §91.119 minimum altitudes en route. (Possible §91.157 special VFR if KPAO ceiling drops further.)

2. **IFR flight with marginal alternate.** KSQL -> KSAC, forecast 1,200 broken with 4 SM at destination, ETA 1500z. Filed alternate KSMF forecast 2,000 broken 5 SM. 172, 36 gallons. Operative rules: §91.103 preflight, §91.167 IFR fuel (dest + alt + 45), §91.169 alternate required (1-2-3 fails) and alternate qualifies (>600-2 / 800-2 forecast), §91.171 VOR check (within 30 days), §91.175 expected approach minimums.

3. **Climb across Sierra requiring oxygen.** KSAC -> KRNO, climb to 13,500 MSL for 45 minutes. Single-pilot. Operative rules: §91.103 preflight, §91.211(a)(1) crew oxygen required (>30 min above 12,500), §91.119(a) emergency-landing-without-undue-hazard floor (terrain), §91.215 Mode C above 10,000 MSL (with 2,500 AGL exception in mountains), §91.159 hemispheric altitude (eastbound 13,500 -> not on the rule; westbound 12,500 must be 12,500 not 13,500).

4. **Night cross-country with passengers in marginal weather.** KPAO -> KSBA, 1900 local departure, forecast clearing en route. Three passengers, one toddler. Operative rules: §91.107 child-under-two lap rule (parent must actively restrain), §91.107 seatbelts and shoulder harnesses for taxi/takeoff/landing, §91.151 night fuel reserve (45 min, not 30), §91.155 night Class G minimums (3 SM, 500/1,000/2,000), §91.119 night ground reference altitude.

5. **Lost comm IFR descending through Class B.** Last clearance: cleared to KOAK, vectors for traffic, expect ILS 28R, expected altitude 5,000. Current altitude 7,000, lost comm at 7,000. Operative rules: §91.185 route AVEF (vectored -> direct to fix; expected ILS 28R; filed flight plan as fallback), §91.185 altitude (highest of: assigned 7,000, expected 5,000, MEA -> MEA varies but assigned 7,000 wins as long as you're within the assigned segment), §91.183 reports (squawk 7600), §91.215 Mode C and ADS-B remain active, §91.175(c) descent below DH/MDA at the missed approach point.

## Hardcover drill -- the cumulative test

Before moving to Week 5, recite the following from memory in under 5 minutes:

1. The §91.113 right-of-way order (six priorities)
2. The §91.155 weather table (Class B / C-D-E < 10,000 / C-D-E >= 10,000 / G day < 1,200 / G day > 1,200 / G night)
3. §91.157 special VFR gates (clearance, day default, 1 SM, clear of clouds)
4. §91.159 cruising altitude rule (above 3,000 AGL, magnetic course, hemispheric)
5. §91.167 IFR fuel sequence (destination + alternate + 45 minutes; alternate dropped if §91.167(b) met)
6. §91.169 1-2-3 rule (1 hour before/after, 2,000 ceiling, 3 SM)
7. §91.169 standard alternate minimums (600-2 precision; 800-2 non-precision)
8. §91.175(c) ten visual references (ALS, threshold, threshold markings, threshold lights, REIL, VASI, TDZ, TDZ lights, runway, runway lights)
9. §91.185 lost-comm route (AVEF) and altitude (highest of: assigned, expected, MEA)
10. §91.211 oxygen altitudes (12,500-14,000 after 30 min crew; 14,000+ crew always; 15,000+ passenger provided)
11. §91.215 transponder triggers (Class A/B/C, Mode C veil 30 NM/10,000, above 10,000 MSL with 2,500 AGL exception)

If any item takes longer than 30 seconds, return to the lesson and re-read it. The capstone in Week 10 expects this list cold.
