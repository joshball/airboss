# Course Research

Deep-dive research into each FAA core topic area and course design decisions. This is where we understand the source material before building lessons.

## Structure

One folder per FAA core topic (A.1-A.13), plus course-wide research docs.

| Folder                        | FAA Topic                                          | AC Section |
| ----------------------------- | -------------------------------------------------- | ---------- |
| A.1_GPS-Automation-TAA/       | Navigating in the 21st Century (GPS/TAA/Automation) | A.1        |
| A.2_SUA-Airspace-Security/    | Security-Related Special Use Airspace               | A.2        |
| A.3_TSA-Requirements/         | TSA: What Flight Instructors Need to Know           | A.3        |
| A.4_Teaching-Safety-Culture/  | How to Teach Effectively and Build Safety Culture   | A.4        |
| A.5_GA-Safety-Trends/         | Current Safety Trends in GA                         | A.5        |
| A.6_Pilot-Deviations/         | Pilot Deviations: Causes and How to Avoid           | A.6        |
| A.7_FAASTeam-WINGS/           | FAASTeam and WINGS in Your Program                  | A.7        |
| A.8_Regulatory-Updates/       | FAA Regulatory, Policy, and Publications Changes    | A.8        |
| A.9_IPC-Flight-Review/        | How to Conduct an Effective IPC and Flight Review    | A.9        |
| A.10_Ethics-Professionalism/  | Ethics and Professionalism                          | A.10       |

Each folder contains:

- `OVERVIEW.md` -- FAA verbatim description, key teaching points, regulatory references, our content links
- Research notes, source material analysis, lesson design ideas (added over time)
