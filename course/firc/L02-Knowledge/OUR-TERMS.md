# Our Terms

Terms we use in this project that are NOT official FAA terminology. These are our design language for describing the course experience. When writing FAA-facing documents (TCO, submission package), use only FAA terms.

---

## Content Types

| Our term       | What it means                                                                                                  | FAA equivalent                     |
| -------------- | -------------------------------------------------------------------------------------------------------------- | ---------------------------------- |
| Scenario       | An interactive tick-driven simulation where the learner makes CFI decisions in real time                       | Part of "lesson presentation"      |
| Tick            | A single decision point within a scenario -- situation + student state + intervention choice                  | No equivalent                      |
| Intervention   | The learner's chosen action at a tick (Ask/Prompt/Coach/Direct/Take Controls)                                 | No equivalent                      |
| Debrief        | Post-scenario review showing the chain of events, scoring, and what could have been different                  | No equivalent (closest: "testing") |
| Micro lesson   | Short teaching content triggered before/during/after a scenario -- builds knowledge that scenarios test         | Part of "lesson presentation"      |
| Knowledge check | Multiple-choice or multiple-select questions assessing retention                                              | "Written test" (SS 13)             |
| Module         | A group of scenarios, micro lessons, and knowledge checks organized by instructional theme                     | "Lesson" or "module" (SS 11.10)    |
| Student model  | Behavioral profile that determines how a simulated student acts (overconfident, timid, freezes under pressure) | No equivalent                      |
| Competency     | A specific teachable skill with observable behaviors (e.g., "Diagnose student state in real time")             | No equivalent                      |

## Time Types

| Our term            | What it means                                                                        | FAA equivalent                             |
| ------------------- | ------------------------------------------------------------------------------------ | ------------------------------------------ |
| FAA-qualified time  | Time that counts toward the 16-hour FIRC requirement and topic minimums              | "Instructional time" / "course hours"      |
| Exploratory time    | Additional practice time (free play, extra drills) that does NOT count toward the requirement | No equivalent -- FAA only tracks required |
| Active engagement   | Time the learner is making decisions, reading, or interacting (not idle)              | No equivalent                              |

## Scoring Terms

| Our term    | What it means                                                                    | FAA equivalent |
| ----------- | -------------------------------------------------------------------------------- | -------------- |
| Recognition | Did the CFI see the problem developing before it became dangerous?               | No equivalent  |
| Judgment    | Did the CFI match their intervention to the severity of the situation?            | No equivalent  |
| Timing      | Did the CFI act within the window where their intervention could make a difference? | No equivalent  |
| Execution   | Did the CFI's intervention actually address the problem?                         | No equivalent  |
| Outcome     | Student Safe / Student at Risk / Incomplete -- result of the scenario            | No equivalent  |

## Course Flow Terms

| Our term       | What it means                                                          | FAA equivalent                              |
| -------------- | ---------------------------------------------------------------------- | ------------------------------------------- |
| Core topic     | One of the 13 required FAA subject areas (A.1-A.13)                    | "Core topic" (AC 61-83K SS 10.2.1)          |
| Elective topic | Optional additional subject area chosen by the provider                | "Elective topic" (AC 61-83K SS 10.2.2)      |
| Call the Ball  | Commitment moment before starting a scenario (from carrier aviation)   | No equivalent                                |
| Immersion tick | Pre-crisis tick that builds situational awareness without scoring      | No equivalent                                |
| Case I/II/III  | Difficulty levels (from carrier landing categories)                    | No equivalent -- FAA doesn't define difficulty |
