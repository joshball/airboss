# Aviation Weather references

Authoritative source documents for the weather course. Citation targets for every lesson.

## Primary FAA sources

- **AC 00-6** -- Aviation Weather. The handbook. Atmosphere, fronts, hazards, products.
- **AC 00-45** -- Aviation Weather Services. Product catalog: METAR, TAF, AIRMET, SIGMET, charts, the lot. Updated as products evolve.
- **AIM Chapter 7** -- Safety of Flight. Operational weather guidance, hazards, pilot responsibilities.
- **FAA Weather Handbook** (FAA-H-8083-28). Newer, expanding -- gradually subsumes AC 00-6.

## Service documentation

- **AWC (Aviation Weather Center)** -- product home pages and documentation at aviationweather.gov
- **NWS** -- radar mosaic, satellite, surface analysis conventions
- **NOAA** -- historical archives (NCEI), reanalysis datasets

## Reference data

- **Iowa State Mesonet** -- archived METAR/TAF data, useful for canonical events corpus
- **AWC archives** -- historical AIRMETs, SIGMETs, prog charts
- **NCEI Storm Events Database** -- famous wx events with timestamps for canonical-events lookup
