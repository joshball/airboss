## Question Writing Guidelines

- **10 questions per section/subsection** -- covers breadth of the sub-topic
- **No true/false** -- per AC 61-83K SS 13.7
- **Multiple-choice (4 options) or multiple-select (4-5 options)**
- **Scenario-based where possible** -- CFI-focused judgment questions
- **Every question cites AC 61-83K** -- explanation references specific section
- **Competency tags** -- use codes from `libs/constants/src/competency.ts`
- **Pool IDs** -- format: `{TopicCode}-M{Module}-{NN}` (e.g., `A7-M5-04`)
- **Question IDs** -- format: `Q-A{Topic}-{NN}` (e.g., `Q-A7-07`), sequential within topic
- **Module assignment** -- follow existing module mapping per topic

## File Naming Convention

```text
test-questions-a.{topic}.{subsection}.md
```

- `a.1.0.md` = main A.1 section
- `a.1.1.md` = A.1.1 sub-section
- `a.6.2.1.md` = A.6.2.1 sub-sub-section
