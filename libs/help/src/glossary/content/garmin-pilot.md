# Garmin Pilot

**Garmin Pilot** is an electronic flight bag (EFB) application for iOS and Android -- Garmin's competitor to [ForeFlight](/reference/glossary/foreflight). It provides charts, flight planning, weather, and in-flight moving-map navigation, and integrates tightly with Garmin panel avionics and portable ADS-B receivers.

For **weather briefing**, Garmin Pilot functions as a flight-planning aggregator: it pulls METARs, TAFs, AIRMETs, SIGMETs, winds aloft, and graphical products from FAA/NWS feeds and overlays them on the planned route, and can file a briefing request through Leidos Flight Service.

The data underneath is the same regardless of which EFB a pilot chooses -- METARs from the same observation network, AIRMETs from the same Aviation Weather Center. The difference between Garmin Pilot, ForeFlight, and [SkyVector](/reference/glossary/skyvector) is the interface, the avionics integration, and the subscription, not the weather itself.

As with any EFB, the planning-side briefing is distinct from in-flight datalink weather -- see the [flight-deck weather displays](/reference/knowledge/wx-flight-deck-weather-displays) node for the in-flight side.

References: **AC 91-78** (Use of Electronic Flight Bags) for the EFB operational framework.
