# Goal

The slice of study you are focused on right now -- e.g. "Pass PPL written by July."

A goal bundles syllabi (and optional ad-hoc knowledge nodes) into one mental destination. You can have multiple active goals; **one is primary**. The Home page's daily CTA references the primary goal.

A goal is a stable intent. It changes when your target changes (you finish PPL, you start IR). It does not change every time you tweak how a session is shaped -- that's the **plan**.
