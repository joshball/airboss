# Plan

The schedule and session shape attached to your study work -- how long, how often, what to focus on, what to skip.

A plan is configuration: session length, default mode (cards / reps / read), focus and skip domains. **One active plan per user.** The session engine reads the active plan to pick what you study next.

A plan changes whenever you tweak how you want to study. The goal it serves stays put.
