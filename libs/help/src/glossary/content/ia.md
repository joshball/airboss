# IA

**Information Architecture.** How the product is organized into pages and sections -- the map.

Airboss's study app collapses into five top-level sections: Home, Learn, Program, Insights, Reference. The IA decides what's a top-level tab vs what's nested. Good IA fits in working memory and matches how the user thinks about the product, not how the backend is structured.
