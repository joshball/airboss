# Syllabus

An **ACS (Airman Certification Standards)** or **PTS (Practical Test Standards)** document. The structured list of **areas of operation**, **tasks**, and **elements** that the FAA publishes for each certificate.

Each qual links to one or more syllabi. A goal scopes which syllabi you're working through. Knowledge nodes link to syllabus tasks so progress on a node rolls up into the qual it serves.
