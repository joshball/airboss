# Glossary drawer

A right-cluster `?` button that opens the glossary as an overlay. Searchable list of every term used in the app. Click an entry to expand its long-form inline.

The drawer reads from the same content as the canonical `/reference/glossary` page. Daily access via the drawer; canonical home is the page.
