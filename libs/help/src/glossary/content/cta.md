# CTA

**Call-To-Action.** The button or link that is the obvious next thing to click on a page.

A page should have one **primary CTA** -- the big, obvious button -- and any number of smaller **secondary CTAs** for follow-on actions. On the study Home, the primary CTA changes with the user's state:

- **No goal:** "Set your first goal."
- **Goal but no plan:** "Build a plan for {goal}."
- **Goal and plan:** "Start today's session."

If you can't tell which button is the CTA on a page, that page has a problem.
