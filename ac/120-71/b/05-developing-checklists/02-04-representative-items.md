5.2.4   Representative Items. Representative items are selected items that represent a whole
        subset of flow actions such that if the selected item was performed, the whole subset must
        have also been performed. For instance, if all engine indications are in the green, the
        whole engine start sequence must have been done correctly; therefore, checking the
        engine indications can serve as a representative item for the engine start sequence.
