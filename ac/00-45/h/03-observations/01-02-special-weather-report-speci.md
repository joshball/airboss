3.1.2   Special Weather Report (SPECI).
        A SPECI is an unscheduled report taken when any of the criteria given in Table 3-1,
        SPECI Criteria, are observed during the period between hourly reports. SPECIs contain
        all data elements found in a METAR. All SPECIs are issued as soon as possible when
        relevant criteria are observed.

        Whenever SPECI criteria are met at the time of the routine METAR, a METAR is issued.
