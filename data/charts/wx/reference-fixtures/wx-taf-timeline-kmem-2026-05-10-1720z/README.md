# wx-taf-timeline-kmem-2026-05-10-1720z

Phase G TAF timeline reference chart for the wx-charts library.

## Source

Raw TAF pulled from the FAA Aviation Weather Center API on 2026-05-10:

```text
TAF KMEM 101720Z 1018/1124 34006KT P6SM SCT050 BKN120
  PROB30 1019/1024 4SM -TSRA BKN045CB
  FM110100 03009KT P6SM VCSH BKN060 BKN100
  PROB30 1102/1104 4SM -TSRA BKN045CB
  FM111200 04009KT P6SM BKN060 OVC100
  FM112000 03009KT P6SM SCT250
```

Source URL: <https://aviationweather.gov/api/data/taf?ids=KMEM&format=raw>

The IEM and AWC archives only retain ~30 days of historical TAFs and the
IEM AFOS PIL endpoint blocks scripted pulls; this fixture is the most
recent real TAF that exercises every change-group kind (FM x3, PROB30 x2,
plus the INITIAL period). KMEM was chosen over the suggested KSEA in the
work-package spec because the KMEM bulletin contains both PROB30 windows
and multiple FM transitions, which lets the timeline demonstrate the full
band/legend palette.
