# NDB Accuracy Check

NDB Accuracy Check
he pilot must identify an NDB before using it for
avigation, and continuously monitor it while using it
or an instrument approach.  The lack of an IDENT may
ndicate that the NDB is out of service, even though it may
till be transmitting (for instance for maintenance or test
purposes). If an incorrect IDENT is heard, then the NDB
should not be used.
