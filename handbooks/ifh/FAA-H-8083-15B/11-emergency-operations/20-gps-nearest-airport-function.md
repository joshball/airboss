# GPS Nearest Airport Function

GPS Nearest Airport Function
Procedures for accessing the nearest airport information vary
by the type of display installed in an aircraft. Pilots can obtain
information relative to the nearest airport by using the PFD,
multi-function display (MFD), or the nearest function on the
GPS receiver. The following examples are based on a popular
system. Pilots should become familiar with the operational
characteristics of the equipment to be used.
