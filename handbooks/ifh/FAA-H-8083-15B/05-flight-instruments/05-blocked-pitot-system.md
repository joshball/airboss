# Blocked Pitot System

Blocked Pitot System
If the pitot tube drain hole becomes obstructed, the pitot
system can become partially or completely blocked. When
dynamic pressure cannot enter the pitot tube opening, the ASI
no longer operates. If the drain hole is open, static pressure
equalizes on both sides of the diaphram in the ASI and the
